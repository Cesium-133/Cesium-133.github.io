建站撒花

大家好，我是目前大一的一个小白博主，这是我的一个技术博客，会分享交流一些学习中的思考

今天特别高兴！忙了两个星期，终于搭建好了我的个人博客 ~

我建站选用的是 Git + Hexo + Github Page + 个人域名 的方式

流程大致是

  1. 购买个人域名，实名认证（阿里云）
  2. 用 Node.js 的 Hexo 来完成本地建站
  3. 用 Git 完成本地文件的上传与版本管理，需要用SSH 公钥和 Github加密连接
  4. 在 Github 上建立一个 Repository，命名为用户名，然后生成一个 Github Page
  5. 把个人域名解析到 Github Page 上
  6. 设置博客的主题等

我也走了很多弯路，比如一开始选用Gitee Page 后发现由于服务器在国内需要备案，很麻烦；比如搞不懂原理从而一头雾水。特别感谢知乎博主 枫叶 的专栏 MyBlog - 知乎 (zhihu.com)，以及广大开源爱好者们，提供了各种开源代码，还手把手帮我一步步建好博客。

不过这个过程学习了很多知识，比如计算机网络原理，一个网站的构成，还有科学上网

年度计划

建这个博客的初心是分享自己学习的一些感受，我希望自己可以每个星期都发表一篇独立思考的博客，可以是学习笔记，可以是生活中的思考等等

之所以想要这样做，是受到费曼学习法的影响。我们在输出知识的时候，特别是用自己的语言来输出的时候，往往可以加深理解和记忆。综合以上，我希望我的博客可以满足以下几个要求

  • 【专业】：专业知识不会出错

  • 【通俗】：尽量通俗易懂，不堆砌专业术语

  • 【形象】：配合图片加强理解记忆（尽量）

  • 【新颖】：只记录自己的理解，或者自己的总结，不会只照抄课本上有的内容，这仅仅作为辅助

听着就觉得很难！但是如果不改变，就一定不会有成长的，对吧
